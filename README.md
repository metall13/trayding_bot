# Bybit Арбитражный Бот

Простой торговый бот на Go для арбитража на бирже Bybit. Бот подключается к WebSocket Bybit, анализирует стакан ордеров и выполняет арбитражные сделки при обнаружении возможностей.

## 🚀 Возможности

- **WebSocket подключение** к Bybit для получения данных в реальном времени
- **Автоматический арбитраж** - поиск и исполнение арбитражных возможностей
- **Telegram уведомления** о всех сделках с расчетом прибыли
- **Риск-менеджмент** - лимиты на размер сделок и проверка баланса
- **Testnet поддержка** - безопасное тестирование на тестовой сети
- **Docker контейнеризация** - простое развертывание

## 📋 Требования

- Docker и Docker Compose
- API ключи от Bybit
- Telegram бот токен

## ⚙️ Настройка

### 1. Получение API ключей Bybit

1. Зарегистрируйтесь на [Bybit](https://www.bybit.com/)
2. Перейдите в раздел API Management
3. Создайте новый API ключ с правами на торговлю
4. **ВАЖНО**: Используйте testnet для тестирования!

### 2. Создание Telegram бота

1. Найдите [@BotFather](https://t.me/botfather) в Telegram
2. Создайте нового бота командой `/newbot`
3. Получите токен бота
4. Узнайте свой Chat ID (можно использовать [@userinfobot](https://t.me/userinfobot))

### 3. Настройка окружения

1. Скопируйте `.env.example` в `.env`:
```bash
cp .env.example .env
```

2. Отредактируйте `.env` файл, указав ваши данные:
```env
BYBIT_API_KEY=your_api_key_here
BYBIT_API_SECRET=your_api_secret_here
TELEGRAM_TOKEN=your_telegram_bot_token_here
TELEGRAM_CHAT_ID=your_telegram_chat_id_here
MAX_TRADE_AMOUNT=10
TESTNET=true
```

## 📊 Как работает бот

1. **Подключение**: Бот подключается к WebSocket Bybit и подписывается на стаканы ордеров для основных торговых пар (BTCUSDT, ETHUSDT, BNBUSDT, ADAUSDT, SOLUSDT)

2. **Анализ**: Для каждой пары бот анализирует стакан ордеров, ищет ордера на продажу (asks) с ценой ниже текущей рыночной цены

3. **Арбитраж**: При обнаружении арбитражной возможности:
   - Размещает лимитный ордер на покупку по выгодной цене
   - Немедленно продает купленный объем по рыночной цене
   - Рассчитывает и отправляет уведомление о прибыли

4. **Риск-менеджмент**:
   - Проверяет доступный баланс перед сделкой
   - Ограничивает размер сделки параметром `MAX_TRADE_AMOUNT`
   - Работает только на testnet для безопасности

## ⚠️ Важные предупреждения

- **Используйте только testnet** для тестирования
- **Начните с малых сумм** даже на testnet
- **Мониторьте работу бота** - арбитражные возможности могут быть кратковременными
- **Проверьте настройки** перед запуском на реальных средствах

## 📈 Мониторинг

Бот отправляет уведомления в Telegram о:
- Запуске и остановке бота
- Каждой выполненной арбитражной сделке
- Ошибках и проблемах

## 🛠️ Настройка параметров

В файле `.env` можно настроить:

- `MAX_TRADE_AMOUNT` - максимальная сумма одной сделки в USDT
- `TESTNET` - использование тестовой сети (true/false)
- Торговые пары настраиваются в коде (функция `connectWebSocket`)
